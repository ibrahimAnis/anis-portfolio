Database Designing Tool



Desinging Database Schema for a project is always fun but writing SQL script is not.Using SQL shell to write SQL script is always hectic.You can generate SQL script but in only one RDBMS language if you use any tool like MySQL Workbench.To bridge these gaps I designed this app.This web based tool provides a canvas to draw database schema and to visualize it.This tool can also generate SQL script in 3 different RDBMS technology.You can download the image of schema and you don't need to install any application on your desktop.
